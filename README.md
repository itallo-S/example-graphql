# back-graphql

## Start do Projeto
Para startar a API fake, antes instale globalmente a dependência `json-server` globalmente e execute:

```bash
  $ json-server --watch example-server.json
  ```
  
Para startar o projeto, execute o comando
```bash
  $ yarn start
```
